# Section quizzes

A short, four-question quiz for each section of the workshop — a self-check that the module's *one
habit* and its key discrimination actually landed, not just that the build went green. Each quiz
opens with the questions and hides the answers behind a collapsible **Answers & discussion** block,
so you attempt first, then check.

The quizzes are grounded in each module's own material (the lab worksheet, the speaker notes, the
gotchas) and favour the *discriminations* the day teaches — post-vs-pre, wiring-vs-field, which
condition gated — over trivial recall. Do the quiz right after the matching lab, while the run is
still fresh.

you just did. Or run them yourself — the answers are one click away in each file.
