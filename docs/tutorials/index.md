# Tutorials

Learning-oriented lessons. If you are new to ConfPlanner, start here: each page takes you by the hand and has you *build and run* something, step by step, with a result you can see. You do not need to make any decisions or understand the framework first — just follow along, and you will end up with a working agent and the confidence to explore on your own.

- **[Getting started](getting-started.md)** — clone the repo, get a green build with no API keys, add one provider key, start the Embabel shell, and produce your very first conference schedule.
- **[Walk the labs](walk-the-labs.md)** — a guided journey through the six lab branches, from typed domain models to model routing, watching the agent grow one habit at a time.

When you finish these and want to *get a specific job done*, move on to the [How-to guides](../how-to/index.md). To *look something up* mid-task, see the [Reference](../reference/index.md). To understand *why* the agent is built this way, read the [Explanation](../explanation/index.md).
